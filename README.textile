Set Up:

Install Ruby 2.1.1:
1. $ curl -L https://get.rvm.io | bash -s
2. $ rvm get stable
3. $ rvm requirements
4. $ rvm install 2.1.1
5. $ rvm use 2.1.1@rails-moodle --create

Install Rails 4.1.1
6. $ gem install rails --version 4.1.0

Set up the Rails app on your localhost:
7. $ git clone git://github.com/sorsaffari/rails-moodle.git
8. $cd rails-moodle
9. $ bundle install
10. $ rake db:migrate
11. $ rake db:seed
12. $ rails generate layout:install bootstrap3
13. $ rails s
